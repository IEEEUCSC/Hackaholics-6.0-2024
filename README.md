# Hackaholics_6.0

## Development

### Setup

**Frontend:**

1. Navigate to the frontend directory:

```bash
cd Frontend
```

2. Install dependencies:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm start
```

**Backend:**

1. Navigate to the backend directory:

```bash
cd Backend
```

2. Install dependencies:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm dev
